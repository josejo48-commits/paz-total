# Paz Total 🕊️✨

App PWA de fe, rutina personal y gestión de empresa — con cielo animado día/noche, sonidos relajantes y versículos bíblicos.

> *«La paz os dejo, mi paz os doy; yo no os la doy como el mundo la da.»* — Juan 14:27

## Características

- **Cielo animado** que cambia según la hora: amanecer 🌅, día ☀️, atardecer 🌇 y noche 🌙 con estrellas en movimiento
- **Sol y luna** con fases reales, rayos animados y efecto de luna creciente
- **Nubes** que flotan y cambian de color según el momento del día
- **Sonidos relajantes**: lluvia, bosque con pájaros, y binaural de calma
- **Versículo bíblico** del día con temas: paz, fuerza, fe, propósito y descanso
- **Rutina diaria** con anillo de progreso animado
- **Gestión de empresa**: tareas, tablero kanban y proyectos
- **Diario de oración** personal
- **PWA instalable** en Android e iOS — funciona sin internet

## 🚀 Publicar en GitHub Pages

### 1. En GitHub — nombra el repositorio: `paz-total`

### 2. Sube estos archivos al repositorio:
```
index.html
manifest.json
sw.js
README.md
icons/
  icon-192.png
  icon-512.png
```

### 3. Activa GitHub Pages
1. Settings → Pages
2. Source: **Deploy from a branch**
3. Branch: **main** / **/ (root)** → Save

### 4. Tu link será:
```
https://josejo48-commits.github.io/paz-total/
```

### 5. Instalar en celular

**Android (Chrome):** Menú ⋮ → *"Añadir a pantalla de inicio"*

**iPhone (Safari):** Compartir □↑ → *"Añadir a inicio"*
